Sports Betting App

View Project

Integrating a Django app with The Odds API, this app shows a list of events within the date range filter. Selecting the sporting event then displays bookmaker odds on that matchup.

  • A GitHub repo and separate staging site for version control.
  • Environmental variables to protect information like API keys.
  • Models that store data to limit the number of API calls, and a Heroku PSQL database to store data.
  • Heroku Scheduled tasks to handle fetching API data at regular intervals.
